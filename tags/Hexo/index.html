使用Hexo搭建个人博客

概念
什么是Hexo？

Hexo是基于Node.js的静态博客程序，可将生成的静态网页托管于Github和Heroku。

什么是Node.js？

Node.js是运行在服务端的JavaScript。

部署至Github Pages

概念
什么是Github？

GitHub是通过Git进行版本控制的软件源代码托管服务平台。

什么是Git？

Git是分布式版本控制工具。

什么是Github Pages？

GitHub Pages是通过GitHub托管和发布的公开网页。

工具安装
git安装
git部署插件安装

注册

Github注册网址

创建仓库

  1. 点击New创建仓库；

  2. Repository name处填入username.github.io，username为Github帐户名称，最后点击Create repository创建仓库；

  3. 记录仓库地址；

添加ssh公钥至Github

  1. 查看是否存在SSH 公钥。

  2. 若不存在，则生成 SSH 公钥。

    邮件地址换成自己的邮件地址，然后一路回车，使用默认值即可。
    因为个人使用，无需设置密码。
    复制输出文本。

  3. 添加ssh公钥至Github。

    • 点击右上角符号，在点击Setting进入设置页面。

    • Setting页面点击SSH and GPG keys。

  • SSH and GPG keys页面点击New SSH keys。

    在Title处填入记号，可以是用户名或者其它。在Key处粘帖输出文本。
