20200309-Ubuntu Server 18.04安装Redmine4.1

在Ubuntu Server 18.04上安装Redmine4.1

需求

Redmine4.1 Ruby 2.3, 2.4, 2.5, 2.6 Rails 5.2

Step1. 安装Ubuntu Server 18.04虚拟机

Step1-1.将Ubuntu Server 18.04默认镜像源换成中科大镜像源

Step2. 准备安装环境

Step2-1. 为Redmine服务创建用户

  1. 添加redmine用户并设置密码。

  2. 配置redmine用户。

  • 禁止redmine用户ssh登录。

  • 安装过程赋予redmine用户root权限（注意，安装结束后需要取消redmine用户的root权限）。

  1. 切换到redmine用户。

Step2-2. 准备Redmine安装程序

  1. 安装git。

  2. 克隆Redmine安装程序。

  3. 检出Redmine4.1分支。

Step2-3. 安装Ruby2.5

  1. 查看可安装的ruby版本。

  2. 更换gem源

Step2-4. 安装rails5.2

Step3. 安装和配置MySQL

Step3-1. 安装MySQL

  1. 安装mysql。

  2. 查看mysql状态。

Step3-2. 配置MySQL

  1. 提高MySQL的安全性。

  2. 为Redmine创建数据库。

  3. 为Redmine配置数据库连接文件。

  4. 启动mysql。

  5. 开机自启动。

参考资料

  • https://dev.mysql.com/downloads/repo/yum/
  • https://dev.mysql.com/doc/mysql-yum-repo-quick-guide/en/
  • https://www.digitalocean.com/community/tutorials/how-to-install-mysql-on-centos-7
  • https://dev.mysql.com/doc/refman/8.0/en/linux-installation-yum-repo.html#yum-repo-select-series
  • https://dev.mysql.com/doc/refman/8.0/en/mysql-secure-installation.html

Step4. 安装Redmine

  1. 安装redmine所需要的依赖。

  2. 生成会话存储密钥。

  3. 创建数据库模式对象。

  4. 设置数据库默认数据。

  5. 设置文件权限。

  6. 试运行，在网页填入192.168.220.144:20001即可浏览。

Step5. 开机启动redmine

  1. 创建redmine.service。

  2. 配置。

Step6. 清理工作

  1. 禁止redmine用户ssh登录。

  2. 取消redmine用户的root权限。
