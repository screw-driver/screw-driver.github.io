Source Insight配置教程

source insight 配置文件放置在百度云。

基本功能

  • 语法高亮
  • 代码提示：类型、变量、函数等。
  • 括号、单引号、双引号之类的自动补全
  • 代码导航：跳转到定义，调用处。

Source Insight使用

  1. 在需要审计的工程根目录下，新建SI目录，用于存储Source Insight产生的文件。

  2. 新建Source Insight工程：Project -> New Project。

  3. 修改工程名和存储路径，点击OK。

  4. 在File Path中选择待审计工程的目录，点击OK。

  5. 在Directory栏中选中工程目录，取消勾选Show only known file types，点击Add Tree；再选中IS目录，点击Remove Tree；最后点击close。
